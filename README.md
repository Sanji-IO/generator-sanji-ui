# generator-sanji-ui

## Install

Setup development enviroment
```sh
wget -qO- https://raw.githubusercontent.com/creationix/nvm/v0.32.1/install.sh | bash
nvm install 6
```

Install generator
```sh
npm install -g yo generator-sanji-ui
```

## Upgrade Generator
```sh
npm update -g generator-sanji-ui
```

## Getting Started

```sh
yo sanji-ui
```
